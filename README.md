Данный робот создан в среде UiPath. Основная его цель - составление отчета об электронных товарах с площадки озон.
Если разобрать подробнее сам процесс: Робот заходит на сайт Ozon с помощью бразуера Chrome,
просит у пользователя ввести количество наименований по которым будет производится поиск,
далее просит ввести сами наименования, после чего осуществляет поиск товаров на площадке Ozon,
считывает данные о товаре: ссылку на товар, наименование, цену со скидкой и цену без скидки, используются только первые 3 страницы результатов поиска.
Вносит все полученные данных в эксель документ, где каждый лист - отдельное наименование, который сохраняет на рабочий стол с указанием даты отчета.
После чего робот выполняет написанный макрос для форматирования таблицы с данными по ширине, выравнивание по центру, форматы столбцов с ценами - денежные.
Заключительным этапом является использование шаблона для отправки SMTP сообщения с помощью mail.ru, в который подставляются необходимые значения аргументов,
само сообщение содержит приветствие и отчет во вложении.
Так же в теле робота отработана возможность возникновения ошибки, например как отсутсвие интернета при открытии браузера:
в таком случае робот попробует трижды перезагрузить страницу, если это удалось, то продолжает работу, если ошибка сохранилась,
то делает скриншот браузера, отправляет уведомление о сбое на почту, которое содержит скриншот и требование проверить ошибки,
удаляет скриншот и закрывает браузер.
